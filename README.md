// Dieses Projekt stellt das klassische Spiel Schere, Stein, Papier dar.

Ziel des Projekts ist es, das Spiel in JavaScript umzusetzen und grundlegende Programmierlogik zu üben. Dabei können Benutzer gegen den Computer antreten und erhalten sofort visuelles Feedback über Sieg, Niederlage oder Unentschieden.

Features

Benutzer wählt Schere, Stein oder Papier

Computer wählt zufällig

Ergebnis wird automatisch berechnet
